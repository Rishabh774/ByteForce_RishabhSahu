## 📝 API Endpoints Live

### Authentication
- POST /auth/register
- POST /auth/login  
- GET /auth/profile
- PUT /auth/profile

### Events
- GET /events
- GET /events/:id
- POST /events
- PUT /events/:id
- DELETE /events/:id

### Bookings
- POST /bookings
- GET /bookings/my-bookings
- PUT /bookings/:id/cancel
- PUT /bookings/verify-attendance

### Ratings
- POST /ratings
- GET /ratings/event/:eventId
- PUT /ratings/:id
- DELETE /ratings/:id
