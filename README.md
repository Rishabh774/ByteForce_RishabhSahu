📌 Project Description
🔴 Problem

In many colleges and local communities, people struggle to discover nearby events like fests, workshops, and meetups.

No centralized platform for local events
Ticket booking is unorganized (Google Forms, WhatsApp, etc.)
No proper attendance management
Event organizers face difficulty in managing registrations
🟢 Solution

Our platform provides a centralized web-based system where users can:

Discover nearby events
Book tickets instantly
Get QR-based entry passes
Manage and track event participation

Event organizers can:

Create and manage events
Track registrations
Verify attendees using QR codes
✨ Key Features
🔐 User Authentication (User & Organizer roles)
🔍 Discover Nearby Events
🎟️ Easy Ticket Booking System
📅 Event Date & Time Management
📊 User Dashboard (My Bookings)
📸 QR Code Ticket Generation
✅ QR-based Attendance Tracking
⭐ Ratings & Feedback (optional)
